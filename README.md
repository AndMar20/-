# SecureMessenger

Небольшое WPF-приложение для обмена зашифрованными сообщениями между двумя экземплярами.

## Запуск

1. Убедитесь, что установлен .NET 8 SDK и запуск выполняется на Windows (для WPF).
2. Откройте файл `SecureMessenger.sln` в Visual Studio 2022+ (решение лежит в корне репозитория, проект — в папке `SecureMessenger`).
2. Соберите и запустите два экземпляра приложения (можно из разных папок `bin`).
3. В каждом окне задайте одинаковый "Общий ключ" и разные порты прослушивания.
4. Нажмите «Запустить сервер», чтобы начать слушать порт.
5. В блоке «Адрес получателя» укажите хост и порт второго экземпляра и отправьте сообщение.

Каждое сообщение шифруется симметрично (AES с ключом, полученным из общего пароля) и передаётся с длиной кадра по TCP.
